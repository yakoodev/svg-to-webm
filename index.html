SVG → WebM

Вставь код SVG, посмотри живое превью с авторазмером, экспортируй WebM и проверь прозрачность на шахматке. CSS keyframes фиксируются по времени, а WebM пишется в реальном времени без замедления. Для максимального качества используй локальный экспорт через Puppeteer + ffmpeg.

Настройки

Превью автоматически берёт размер из width/height или viewBox. Для качества лучше экспортировать в 2× или 3× и уменьшать размер видео через CSS. В v7 браузерный экспорт пишет видео в реальном времени: если SVG тяжёлый, кадры могут пропускаться, но скорость WebM остаётся правильной. Если нужен настоящий production-файл с alpha и максимально чистые края, используй npm run export:svg.

Если на шахматке прозрачные места стали чёрными — браузерный MediaRecorder потерял alpha. Тогда экспортируй локально через export-node.

    


    

      Команда для качественного локального экспорта:
      npm run export:svg -- examples/yakoo_pnv_animated_banner_smooth_intro.svg output.webm --fps=60 --strip-bg --scale=2 --crf=10
      

      После экспорта рядом создаётся output.preview.html — открой его, чтобы проверить прозрачность WebM на шахматке/чёрном/белом фоне.
